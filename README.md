# Python3 Flask Hello

Learning flask with Head First Python 2nd Edition

## 1. Install

- Install python3
```bash
sudo apt install python3.8
```

- Install flask
```bash
pip3 install flask
pip3 install Jinja2
```

## 2. Run

- Run flask app
```bash
flask run
# or
python3 -m flask run
```

- Access http://127.0.0.1:5000/
